## What is in this repository?

This repository hosts the original documents/dataset artefacts used in our study on generating UML sequence diagrams from natural language.

- `Original documents/`  
  Contains the source materials used to construct the dataset.  
  Note: The exact file formats and organisation may vary by domain/source. Please inspect the folder for details.
